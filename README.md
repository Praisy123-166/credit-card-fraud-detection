# credit-card-fraud-detection
Random Forest &amp; Neural Network classifier detecting fraudulent transactions from 284,807 real credit card records | 95.28% AUC | Python, scikit-learn
Machine learning model to detect fraudulent financial transactions 
using a real-world dataset of 284,807 credit card transactions.

## Results
- AUC Score: 95.28%
- Fraud Precision: 94%
- Fraud Recall: 83%

## Tools Used
Python, Pandas, NumPy, scikit-learn, Matplotlib, Seaborn

## What I built
- Random Forest classifier on highly imbalanced financial data
- Stratified train/test split to handle 0.17% fraud rate
- Confusion matrix and feature importance visualizations

## Dataset
Kaggle — Credit Card Fraud Detection (MLG-ULB)
